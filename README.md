# Spec-First Methodology — Claude Code Skill

Скилл для Claude Code и Cowork, который проводит через методологию **Spec-First** — подход к AI-driven разработке, при котором проект полностью описывается в документации **до** написания первой строки кода.

> Спецификация — это ТЗ для AI-агентов, а не для людей. Чем конкретнее описаны таблицы, эндпоинты, компоненты — тем автономнее работает Claude Code.

---

## Что умеет скилл

- **Слой 0 — Клиентский бриф**: генерирует кастомную анкету под домен заказчика (салон красоты, SaaS, e-commerce и т.д.)
- **Слой 1 — PROJECT_IDEA.md**: структурирует идею в 10 разделов для AI
- **Слой 2 — Техническая спецификация**: описывает каждый модуль так, что субагент реализует его без уточнений
- **Слой 3–4 — Конфигурация**: генерирует CLAUDE.md (до 120 строк), субагентов, rules, skills
- **Слой 5 — Промпт-инструкции**: управляющие документы для переходов между слоями

## Структура

```
spec-first/
├── SKILL.md
└── references/
    ├── templates.md   — кастомные брифы по доменам
    └── checklist.md   — чеклист готовности к запуску
```

## Установка

### Для Cowork
Скачай `spec-first.skill` из Releases и установи как плагин.

### Для Claude Code (VS Code / терминал)
Скопируй папку в проект:
```
твой-проект/
└── .claude/
    └── skills/
        └── spec-first/
            ├── SKILL.md
            └── references/
```
